Lesson 24.5.  Overview of conditions, 3: Contrary-to-fact conditions

 

The presentation of conditions started in the second semester with 13.3, 14.4, and 16.4.  The first chart given this semester, Overview 1 =  20.4, included only Types I, II, and  V(partially), studied earlier.  Overview 2 = 22.4 added Type III ("should/would" conditions).  This Overview 3 adds Type IV = "Contrary-to-fact" or "Unreal" conditions.  They are dubbed thus because it is assumed that the condition is not fulfilled (Contrary-to-fact in the present) or was not fulfilled (Contrary-to-fact in the past) .

In Greek the Type IV examples given in the chart would read:

a) εἰ ἦς / ἦσθα[1] προφήτης, προύλεγες[2]  ἂν τὰ μέλλοντα.

b) εἰ μὴ ἔδραμε, οὐκ ἂν ἔπεσε.

 

In many instances, the combination  (εἰ  + Imperfect ) [ Imperfect +  ἄν  ]  expresses a condition that is not fulfilled now, while  (εἰ  + Aorist ) [ Aorist +  ἄν ]  refers to a condition that was not fultilled in the past ... but this is not necessarily always the case, despite what traditional grammars may say.   Attention has been called recently to the ambiguity of these constructions.   Context and sense will guide us to interpret each text.  As a starting point, however, it is useful to translate thus:

 

Type IV a) conditional period = if something were true now //but it isnt//, something else would take place now //but doesnt//,

                                                                                             and  

Type IV b) conditional period = if something had happened in the past //but it didnt//, something else would have taken place //but didnt//.

 

More examples

1) οὐδεὶς τῶν ἀρχόντων τοῦ αἰῶνος τούτου ἔγνωκεν //θεοῦ σοφίαν//, (εἰ γὰρ ἔγνωσαν,)  [οὐκ ἂν τὸν κύριον τῆς δόξης ἐσταύρωσαν.]  [I Cor 2:8]

None of the princes of this world knew //the wisdom of God//: for had they known it, they would not have crucified the Lord of glory.

 

2) (εἰ δὲ ἑαυτοὺς διεκρίνομεν,) [οὐκ ἂν ἐκρινόμεθα·]  [1 Cor 11:31]

For if we  judged ourselves, we would not be judged.

 

3) εἰ δὲ ἦν τὰ πάντα ἓν μέλος, ποῦ τὸ σῶμα;  [1 Cor 12:19]

If all were a single member, where //would be// the body?

 

4) εἰ ἔτι ἀνθρώποις ἤρεσκον, Χριστοῦ δοῦλος οὐκ ἂν ἤμην.  [Gal 1:10]

If I were still pleasing people, I would not be a servant of Christ.

 

 


[1]  The second form is typical of koinê Greek.

[2]  This is the Imperfect of  προ-λέγω, with contraction: προέλεγον > προύλεγον
